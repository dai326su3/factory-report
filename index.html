Googleスプレッドシート連携手順

以下の手順でGoogleスプレッドシートと連携することで、入力データが自動的に収集・集計されます。

  • Googleドライブを開き、新しいスプレッドシートを作成する
  • メニューの「拡張機能」→「Apps Script」を開く
  • 以下のコードをコピーして貼り付け、保存する（Ctrl+S）
  • 「デプロイ」→「新しいデプロイ」→「ウェブアプリ」を選択
  • 「アクセスできるユーザー」を「全員」にしてデプロイ
  • 表示されたURLをコピーして「入力」タブのURL欄に貼り付け、保存する

通知メールアドレス

報告送信時にメール通知を送るアドレスを登録してください。複数登録できます。

Apps Script コード

このコードをApps Scriptに貼り付けてください：

function doPost(e) { try { var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet(); if (sheet.getLastRow() === 0) { sheet.appendRow(['発生日時','発生場所','不適合種類','詳細','発見者名','対処状況','送信日時']); } var data = JSON.parse(e.postData.contents); sheet.appendRow([ data.datetime, data.location, data.type, data.detail, data.reporter, data.status, new Date().toLocaleString('ja-JP') ]); ``` // メール通知 if (data.sendEmail && data.emails && data.emails.length > 0) { var subject = '【社内不適合報告】' + data.type + ' - ' + data.location; var body = '社内不適合が報告されました。\n\n' + '発生日時：' + data.datetime + '\n' + '発生場所：' + data.location + '\n' + '種類：' + data.type + '\n' + '詳細：' + data.detail + '\n' + '発見者：' + data.reporter + '\n' + '対処状況：' + data.status + '\n'; MailApp.sendEmail(data.emails.join(','), subject, body); } return ContentService.createTextOutput(JSON.stringify({result:'success'})) .setMimeType(ContentService.MimeType.JSON); ``` } catch(err) { return ContentService.createTextOutput(JSON.stringify({result:‘error’,message:err.toString()})) .setMimeType(ContentService.MimeType.JSON); } }
